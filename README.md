# 📝 Routing 구조

1. "/" : 책 목록 페이지 구현
2. "/bookDetail/[id]" : 책 상세 정보 페이지/뷰 구현
3. "/bookDetail/[id]/edit" : 책 정보 수정 페이지 구현
4. "/addBook" : 책 추가 페이지 구현
